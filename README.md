# Digital Wallet System

A full-stack Digital Wallet application built using **Spring Boot (Backend)** and **HTML/CSS/JavaScript (Frontend)**.

## Features
- User wallet creation
- Add / deduct balance
- Transaction handling
- Simple and responsive UI
- REST API based backend

## Tech Stack
**Frontend:**
- HTML
- CSS
- JavaScript

**Backend:**
- Java
- Spring Boot
- REST APIs

## Project Structure

digital-wallet-system/
│
├── frontend/ # UI files (HTML/CSS/JS)
├── backend/ # Spring Boot application


## How to Run

### Backend

cd backend
mvn spring-boot:run


### Frontend
Just open:

frontend/index.html


## API Endpoints (example)
- GET /wallet
- POST /wallet/add
- POST /wallet/withdraw
